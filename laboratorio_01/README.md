# Resumen del Ecosistema de Ciencia de Datos

Este repositorio contiene el proyecto final del curso "Herramientas para la Ciencia de Datos".

## Sobre el Proyecto

El archivo principal, `DataScienceEcosystem.ipynb`, es un Jupyter Notebook creado como parte de la asignación del curso. Este cuaderno incluye:

* Un resumen de los lenguajes, bibliotecas y herramientas populares utilizados en la Ciencia de Datos.
* Ejemplos de celdas de código que ejecutan expresiones aritméticas básicas en Python.
* Demostraciones del uso de celdas de Markdown, incluyendo la creación de listas (ordenadas y desordenadas), tablas y texto con formato.

## Herramientas Utilizadas

* **Jupyter Notebook:** Para la creación y ejecución del cuaderno.
* **Python:** Como lenguaje de programación principal para las celdas de código.
* **GitHub:** Para alojar y compartir el proyecto final.
